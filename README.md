# Dart Board

Rough Outline

- Piezo sensors on back of cork board (decide number of sensors later)
- One PCB, using surface mount components
- Battery powered, can just use USB-C input power for now
- Use I2C bus to communicate with ADC chips
- on/off behavior: switch on the outside of the case, on the backside of the PCB

- Current goal: prove working concept by having functional devboard-sensor-ADC compatability
    - Prototype PCB with board to adc to piezo connection for OPS2 design review
